Calidad de Software

Calidad de software se refiere al grado en que un software cumple con los requisitos funcionales y no funcionales esperados, así como con las necesidades del usuario final. Abarca tanto el funcionamiento técnico del software como la experiencia del usuario al utilizarlo.

Etiqueta strong

Etiqueta i

Define un texto en cursiva de un documento HTML, que se utiliza para marcar una parte del texto como diferente o especial, como un término técnico, un idioma extranjero o un pensamiento.

Etiqueta em

Define un texto enfático de un documento HTML, que se utiliza para expresar el énfasis o el tono de una parte del texto.

Etiqueta mark

Define un texto marcado de un documento HTML, que se utiliza para resaltar o señalar una parte del texto, como una coincidencia de búsqueda o una referencia.

Etiqueta small

Define un texto pequeño de un documento HTML, que se utiliza para mostrar información secundaria, como una nota al pie, una fecha o una hora.

Etiqueta del

Etiqueta ins

Etiqueta sub
